Open-Source Distribution :h3

The fftMPI library is licensed under the modified "Berkeley Software
Distribution (BSD) License"_bsd.  This is a very non-restrictive
license which basically means the library can be used by anyone for
any purpose, including commercial ones.

The legal text for the license is in the LICENSE file included in the
fftMPI distribution.  As stated there, the fftMPI library comes with
no warranty of any kind.  The copyright notice is listed in the header
of each source file.

See "www.gnu.org"_gnuorg or "www.opensource.org"_opensource for a
discussion of the open source philosophy.
